The geometry is isotropic because it contains rotations about each point. Indeed, the group \(\cal S\) contains all elliptic Möbius transformations fixing antipodal points \(p\) and \(p_a\text{.}\) These Möbius transformations rotate points (by any angle \(\theta\)) about type II clines of \(p\) and \(p_a\text{.}\)
