Definition 5.3.6.

If \(\boldsymbol{r}: [a,b] \to \mathbb{D}\) is a smooth curve in the hyperbolic plane, define the length of \(\boldsymbol{r}\), denoted \({\cal L}(\boldsymbol{r})\text{,}\) to be

\begin{equation*} {\cal L}(\boldsymbol{r}) = \int_a^b \frac{2}{1 - |\boldsymbol{r}(t)|^2}~|\boldsymbol{r}^\prime(t)|dt\text{.} \end{equation*}
