  • In \((X_k,G_k)\text{,}\) the distance from 0 to \(x\) is given by

    \begin{equation*} d_k(0,x) = \begin{cases}s \ln\bigg(\frac{s+x}{s-x}\bigg) \amp \text{ if \(k \lt 0\); } \\ 2x \amp \text{ if \(k=0\); } \\ 2s \arctan(x/s) \amp \text{if \(k \gt 0\).} \end{cases} \end{equation*}

  • A circle with radius \(r\) as measured in \((X_k,G_k)\) has area \(A(r)\) given by

    \begin{equation*} A(r) = \begin{cases} 4\pi s^2 \sinh^2(\frac{r}{2s})\amp \text{ if \(k \lt 0\); } \\ \pi r^2 \amp \text{ if \(k=0\); } \\ 4\pi s^2 \sin^2(\frac{r}{2s}) \amp \text{if \(k \gt 0\),} \end{cases} \end{equation*}

  • A circle with radius \(r\) as measured in \((X_k,G_k)\) has circumference \(C(r)\) given by

    \begin{equation*} C(r) = \begin{cases}2\pi s \sinh(r/s) \amp \text{ if \(k \lt 0\); } \\ 2\pi r \amp \text{ if \(k=0\); } \\ 2\pi s \sin(r/s) \amp \text{if \(k \gt 0\).} \end{cases} \end{equation*}
