Solution 2.2.4.1.

If \(z = re^{i\theta}\) then \(z = r(\cos(\theta) + i\sin(\theta))\) so that \(\overline{z} = r(\cos(\theta) - i\sin(\theta))\text{.}\) Since \(\cos(-\theta) = \cos(\theta)\) and \(\sin(-\theta) = -\sin(\theta)\) it follows that \(\overline{z} = r(\cos(-\theta)+i\sin(-\theta))\) so that \(\overline{z} = re^{-i\theta}\text{.}\)
