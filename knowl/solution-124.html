Solution 5.4.6.1.

If two sides of a hyperbolic triangle are Euclidean segments, then the included angle can be calculated using the formula for \(\angle uvw\) from Section 2.3. For instance, the angle at 0 is \(45^\circ\text{.}\)

To find the angle at another corner, we first transform the triangle so that the corner is moved to the origin (since angles are preserved by maps in \(\cal H\) this is fine to do). A Sage cell is handy here. We can use the following Sage code to build a map in \(\cal H\) that moves 1/2 to the origin, and tracks the image of the other two points. Then we can compute the angle of the corner using \(\angle uvw = \text{arg}((w-v)/(u-v)).\)

The angle at 1/2 is about \(71.57^\circ\text{,}\) at \(\frac{1}{2}+\frac{1}{2}i\) it is about \(26.57^\circ\) so the triangle has area

\begin{equation*} A \approx \pi - \frac{\pi}{180}(45+71.57+26.57) = 0.64 {\rm ~units}^2\text{.} \end{equation*}
