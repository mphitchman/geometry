Then the stereographic projection map \(\phi:\mathbb{S}^3 \to \widehat{\mathbb{R}^3}\) is given by

\begin{equation*} \phi((a,b,c,d)) = \begin{cases}\frac{a}{1-d}\textbf{i}+\frac{b}{1-d}\textbf{j}+\frac{c}{1-d}\textbf{k} \amp \text{ if \(d \neq 1\); } \\ \infty \amp \text{ if \(d=1\). } \end{cases} \end{equation*}
