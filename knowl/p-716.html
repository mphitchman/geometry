By definition, \(d(x,y) \geq 0\) for all \(x,y\) in \(S\) and \(d(x,y)=0\) if and only if \(x = y\text{.}\) For the second condition, \(d(x,y) = d(y,x)\) for all \(x,y\) in \(S\) since both distances are either 1 (if \(x \neq y\)) or 0 (if \(x = y\)). Finally, suppose \(x,y,z\) are in \(S\text{.}\) If \(x = y = z\) then both \(d(x,z) = 0\) and \(d(x,y)+d(y,z) = 0\) and the result holds. Otherwise, not all three points are equal, which means \(d(x,z) \leq 1\) and \(d(x,y)+d(y,z) \geq 1\) so that the result holds.
