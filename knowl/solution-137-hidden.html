1. It is remarkably convenient to transfer the situation to the upper half-plane. By a rotation in \(\mathbb{D}\) assume the ideal point common to both horocycles is \(i\text{.}\) Then \(V\) takes the horocycles to parallel, horizontal Euclidean lines in \(\mathbb{U}\) and the other legs to vertical Euclidean lines, so that the region is a Euclidean rectangle. We assume that the rectangle is determined by \(a \leq x \leq b\) and \(m \leq y \leq n\) as pictured.
  2. The segments \(pq\) and \(st\) get mapped by \(V\) to the two vertical legs. These legs are described by the curves
    \begin{equation*} \boldsymbol{r}_3(t) = a + ti~~\text{ for }~ m \leq t \leq n \end{equation*}
    and
    \begin{equation*} \boldsymbol{r}_4(t) = b + ti~~\text{ for }~ m \leq t \leq n\text{.} \end{equation*}
    The lengths of these curve are easily shown to be equal, with
    \begin{align*} {\cal L}(\boldsymbol{r}_3) \amp = {\cal L}(\boldsymbol{r}_4)\\ \amp = \ln\left(\frac{n}{m}\right)\text{.} \end{align*}
  3. To compute the side lengths, let
    \begin{equation*} \boldsymbol{r}_1(t) = t + mi ~~\text{ for }~ a \leq t \leq b \end{equation*}
    and
    \begin{equation*} \boldsymbol{r}_2(t) = t + ni ~~\text{ for }~~ a \leq t \leq b \end{equation*}
    describe the two horizontal segments in the block. Following the statement of the problem, set \(c = {\cal L}(\boldsymbol{r}_1)\) and \(d = {\cal L}(\boldsymbol{r}_2)\text{.}\) Note that \(\boldsymbol{r}_1^\prime(t) = \boldsymbol{r}_2^\prime(t) = 1\text{,}\) so
    \begin{equation*} c = \int_a^b \frac{1}{m}~dt = \frac{b-a}{m} \end{equation*}
    and
    \begin{equation*} d = \int_a^b \frac{1}{n}~dt = \frac{b-a}{n}\text{.} \end{equation*}
    Let \(x = \ln(n/m)\) denote the common length from part (b) so that \(e^x = \frac{n}{m}\text{.}\) Then
    \begin{align*} d e^x \amp= \frac{b-a}{n}\cdot \frac{n}{m}\\ \amp= \frac{b-a}{m}\\ \amp= c \end{align*}
    as desired.
  4. The area of the block is simply the difference of the two horizontal leg lengths:
    \begin{align*} A \amp= \int_a^b\int_m^n \frac{1}{y^2}~dy~dx\\ \amp= (b-a)(\frac{1}{m}-\frac{1}{n}) \\ \amp= \frac{b-a}{m}-\frac{b-a}{n}\\ \amp= c - d\text{.} \end{align*}
