Solution 3.4.4.1.

Given \(T(z) = (az+b)/(cz+d)\text{,}\) let \(T_k(z) = (kaz+kb)/(kcz+kd)\) and note that \(T\) and \(T_k\) define the same transformation of \(\mathbb{C}^+\text{.}\) Note further that \(\text{Det}(T_k)= k^2(ad-bc).\) So, \(T_k\) has determinant one if and only if we set

\begin{equation*} k = \frac{\pm 1}{\sqrt{ad-bc}}. \end{equation*}

That is, the form for \(T\) that has determinant 1 is unique up to sign.
