Solution 3.5.3.1.

Suppose \(T(T(z)) = z\text{.}\) If \(T\) isn't the identity then \(T\) fixes one or two points. If \(T\) fixes two points, it has normal form

\begin{equation*} \frac{T(z) - p}{T(z)-q}=\lambda\frac{z-p}{z-q}\text{.} \end{equation*}

Plug in \(T(z)\) for \(z\) to see

\begin{equation*} \frac{z-p}{z-q} = \lambda\frac{T(z) - p}{T(z)-q} \end{equation*}

which implies that

\begin{equation*} \frac{z-p}{z-q} = \lambda\bigg[\lambda\frac{z-p}{z-q}\bigg] \end{equation*}

which means that \(\lambda^2 = 1\) so that \(|\lambda| = 1\) and \(T\) is elliptic.

Can \(T\) fix just one point? If so, \(T\) has normal form

\begin{equation*} \frac{1}{T(z) - d} = \frac{1}{z-p} + d\text{.} \end{equation*}

Plug in \(T(z)\) for \(z\) to see that

\begin{equation*} \frac{1}{z-p} = \frac{1}{T(z)-p}+d\text{,} \end{equation*}

and these two equations combine to show \(d = 0\) which is not possible since \(T\) isn't the identity map.
