Solution 5.4.9.1.
  1. The center of the circle containing hyperbolic segment \(v_0v_1\text{,}\) denoted \(z_0\text{,}\) has real component equal to \(b\text{,}\) as indicated in FigureÂ 5.4.22. The point \(b\) is the midpoint of Euclidean segment \(v_0 v_0^*\text{,}\) where \(v_0 = r\) and so \(v_0^* = \frac{1}{r}\text{.}\) Thus
    \begin{equation*} b = \frac{r + \frac{1}{r}}{2} = \frac{1+r^2}{2r}\text{.} \end{equation*}
    Since \(\angle b0z_0 = \pi/8\) it follows that the imaginary component of \(z_0\) is \(b \cdot \tan(\pi/8)\text{.}\) That is,
    \begin{equation*} z_0 = \frac{1+r^2}{2r} + \frac{1+r^2}{2r}\tan(\pi/8)i\text{.} \end{equation*}
  2. Refer to the following diagram. The angle at \(v_0\) in the octagon is \(\pi/4\) if and only if the angle \(\alpha = \angle v_1 v_0 0 = \pi/8\text{.}\) But \(\alpha = \pi/8\) if and only if angle \(\tau\) in the figure equals \(\pi - \pi/2 - \pi/8 = 3\pi/8\text{.}\) But right triangle \(\Delta v_0 b z_0\) ensures that \(\tau = 3\pi/8\) if and only if \(\beta = \pi/8\text{.}\) Since \(\beta = \angle bz_0v_0\) we have proved the result.
  3. Consider the right triangle \(\Delta v_0 b z_0\text{.}\) We know that \(\angle bz_0v_0 = \pi/8\) so
    \begin{equation*} \tan(\pi/8) = \frac{|b-v_0|}{|z_0-b|}=\frac{\frac{1+r^2}{2r} - r}{\frac{1+r^2}{2r}\tan(\pi/8)}\text{.} \end{equation*}
    We solve this equation for \(r\) to obtain the result.
  4. \begin{align*} \tan^2(\pi/8) \amp= \frac{\frac{1+r^2}{2r} - r}{\frac{1+r^2}{2r}} \\ \tan^2(\pi/8) \amp= \frac{1 + r^2 - 2r^2}{1+r^2} \\ \tan^2(\pi/8) \amp= \frac{1 - r^2}{1+r^2} \\ \tan^2(\pi/8) + \tan^2(\pi/8) r^2 \amp= 1 - r^2 \\ (1+\tan^2(\pi/8))r^2 = 1 - \tan^2(\pi/8) \\ r^2 = \frac{1 - \tan^2(\pi/8)}{1 + \tan^2(\pi/8)}. \end{align*}
    Using the trig identity
    \begin{equation*} \tan^2(x) = \frac{1-\cos(2x)}{1+\cos(2x)} \end{equation*}
    it follows that
    \begin{equation*} \tan^2(\pi/8) = \frac{2-\sqrt{2}}{2 + \sqrt{2}} \end{equation*}
    so that
    \begin{align*} r^2 \amp= \frac{1 - \frac{2-\sqrt{2}}{2 + \sqrt{2}}}{1+\frac{2-\sqrt{2}}{2 + \sqrt{2}}} \\ \amp= \frac{2+\sqrt{2}- (2-\sqrt{2})}{2+\sqrt{2} + (2-\sqrt{2})}\\ \amp= \frac{2\sqrt{2}}{4}\\ \amp= \bigg(\frac{1}{2}\bigg)^{\frac{1}{2}}. \end{align*}
    It follows that
    \begin{equation*} r = \bigg(\frac{1}{2}\bigg)^{\frac{1}{4}}\text{.} \end{equation*}
