Solution 2.1.4.1.

  1. If \(z = x + yi\) then \(z^2 = (x^2 - y^2)+2xyi\text{.}\) So

    \begin{align*} a \amp = {\text Re}(z^2) = x^2 - y^2 \\ b \amp = {\text Im}(z^2) = 2xy \\ c \amp = z\cdot \overline{z} = |z|^2 = x^2 + y^2 \text{.} \end{align*}

    Now, \(a^2 + b^2 = (x^2 - y^2)^2 + (2xy)^2\) and one shows by expansion and simplification that this equals \((x^2 + y^2)^2\text{,}\) which is \(c^2\text{.}\)

  2. We want \(z = x+yi\) such that \(x^2 - y^2 = 3\text{;}\) \(2xy = 4\text{;}\) and \(x^2 + y^2 = 5\text{.}\) The point \(z = 2 + i\) generates the famous triple \((3,4,5)\text{.}\)
  3. The point \(z = 3 + 2i\) generates \((5,12,13)\text{.}\)
  4. Here are a few: \(z = 5 + 2i\) generates \((21,20,29)\text{;}\) \(z = 4 + 3i\) generates \((7,24,25)\text{;}\) \(z = 5 + 4i\) generates \((9,40,41)\text{.}\)
