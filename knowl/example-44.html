Example5.4.7The length of a curve

To find the length of the horizontal curve \(\vec{r}(t) = t + ki\) for \(a \leq t \leq b\text{,}\) note that \(\vec{r}\hskip.04in^\prime(t) = 1\) and \(\text{ Im}(\vec{r}(t)) = k\text{.}\) Thus, \begin{equation*} {\cal L}(\vec{r}) = \int_a^b \frac{1}{k}~dt = \frac{b-a}{k}. \end{equation*}
