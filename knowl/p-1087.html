The image of the circle on \(\mathbb{S}^2\) defined by \(z = 1/2\) is the circle \(x^2 + y^2 = 3.\) This follows from the comments following TheoremÂ 6.1.8. In particular, the circle on \(\mathbb{S}^2\) determined by the plane \(Ax + By + Cz + D = 0\) gets sent by stereographic projection to the cline

\begin{equation*} (C + D)(x^2 + y^2) + 2Ax + 2By + D - C = 0\text{.} \end{equation*}

Given the plane \(z = 1/2\text{,}\) we have \(A = B = 0\text{,}\) \(C = 1\text{,}\) and \(D = -1/2\text{.}\) Its image under stereographic projection is thus

\begin{equation*} \frac{1}{2}(x^2 + y^2) - \frac{3}{2} = 0\text{,} \end{equation*}

and the result follows.
