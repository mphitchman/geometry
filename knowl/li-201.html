Item 5.4.9.b.
Refer to the following diagram. The angle at \(v_0\) in the octagon is \(\pi/4\) if and only if the angle \(\alpha = \angle v_1 v_0 0 = \pi/8\text{.}\) But \(\alpha = \pi/8\) if and only if angle \(\tau\) in the figure equals \(\pi - \pi/2 - \pi/8 = 3\pi/8\text{.}\) But right triangle \(\Delta v_0 b z_0\) ensures that \(\tau = 3\pi/8\) if and only if \(\beta = \pi/8\text{.}\) Since \(\beta = \angle bz_0v_0\) we have proved the result.
