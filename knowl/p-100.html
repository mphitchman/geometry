Per the following diagram, the angle at \(A\) is \(\alpha\text{,}\) the angle at \(B\) is \(\beta\text{,}\) and the angle at \(C\) is \(\gamma = \gamma_1+\gamma_2\text{.}\) Let \(O\) denote the center of the cone. The quadrilateral \(ABCC^\prime\) in the following diagram has angle sum \(2\pi\text{.}\) Thus, labeling angles as in the diagranm,

\begin{equation*} \alpha + \beta + \gamma_1 + \gamma_2 + \eta_1+\eta_2 = 2\pi \tag{*}\text{.} \end{equation*}

We also know that \(\theta + \phi = 2\pi\) and that \(\eta_1 + \eta_2 + \phi = \pi\) from which we deduce \(\eta_1 + \eta_2 = \pi - \phi = \theta - \pi\text{.}\) Plugging this into (*) we have

\begin{equation*} \alpha + \beta + \gamma = 3\pi - \theta\text{.} \end{equation*}

We note that \(\eta_1 = \eta_2\) because \(\Delta OCC^\prime\) is isosceles, but we do not need to use that fact in the proof.
