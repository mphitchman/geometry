Letting \(\alpha = a + bi\) and \(z = x + yi\text{,}\) the cline equation \(cz\overline{z} + \alpha z + \overline{\alpha} \overline{z} + d = 0\) can be written as

\begin{equation*} c(x^2+y^2)+[ax - by + (ay+bx)i]+[ax-by - (ay+bx)i] + d = 0 \end{equation*}

which simplifies to

\begin{equation*} c(x^2+y^2) + 2(ax - by) + d = 0. \end{equation*}
