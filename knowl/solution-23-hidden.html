Using the quadratic formula,

\begin{align*} z \amp = \frac{-i \pm \sqrt{i^2 + 4(2+6i)}}{2} \\ \amp = \frac{-i \pm \sqrt{7+24i}}{2} \text{.} \end{align*}

Now we find the cartesian form of \(\sqrt{7+24i} = a + bi\text{.}\) Note that

\begin{equation*} 7 + 24i = (a+bi)^2 = (a^2-b^2)+2abi\text{.} \end{equation*}

So we have two equations and two unknowns:

\begin{align*} a^2 - b^2 \amp = 7\\ 24 \amp = 2ab\text{.} \end{align*}

Solve this system to find \(a = 4, b = 3\) (or \(a = -4, b = -3\)).

Thus,

\begin{equation*} z = \frac{-i \pm (4+3i)}{2}\text{,} \end{equation*}

and we have two solutions:

\begin{equation*} z=2+i \text{ or } z = -2 -2i\text{.} \end{equation*}
