Corollary 5.3.10.

The hyperbolic distance function is a metric on the hyperbolic plane. In particular, for any points \(p,q,u\) in \(\mathbb{D}\)

  1. \(d_H(p,q) \geq 0\text{,}\) and \(d_H(p,q)=0\) if and only if \(p = q\text{;}\)
  2. \(d_H(p,q) = d_H(q,p)\text{;}\) and
  3. \(d_H(p,q)+d_H(q,u) \geq d_H(p,u)\text{.}\)
