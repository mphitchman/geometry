So we've established two facts: (1) \(\cal A\) is a subset of \(\cal D\text{,}\) and (2) \(\cal A\) is an invariant set. Since \(\cal D\) is a minimally invariant set it follows by definition that \({\cal A} = {\cal D}\text{.}\) This means that the given set \(B\text{,}\) which is in \(\cal D\text{,}\) is also in \(\cal A\text{.}\) That is, \(A \cong B\text{.}\)
