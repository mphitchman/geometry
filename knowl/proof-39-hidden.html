Suppose \(T\) is a Möbius transformation that preserves antipodal points. If \(T\) is not the identity map then it must fix one or two points. However, if \(T\) preserves antipodal points and fixes a point \(p\text{,}\) then it must fix its antipodal point \(p_a\text{.}\) Thus, \(T\) must have two fixed points and \(T\) has normal form

\begin{equation*} \frac{T(z)-p}{T(z)-p_a} = \lambda\frac{z-p}{z-p_a}. \end{equation*}

To show \(T\) is an elliptic Möbius transformation, we must show that \(|\lambda| = 1\text{.}\) Setting \(z = 0\) the normal form reduces to

\begin{equation*} \frac{T(0) - p}{T(0)-p_a} = \lambda\frac{p}{p_a}. \end{equation*}

Solve for \(T(0)\) to obtain

\begin{equation*} T(0) = \frac{pp_a - \lambda pp_a}{p_a - \lambda p}. \end{equation*}

On the other hand, setting \(z = \infty\) and solving for \(T(\infty)\text{,}\) one checks that

\begin{equation*} T(\infty) =\frac{p-\lambda p_a}{1-\lambda}. \end{equation*}

Since \(T\) preserves antipodal points, \(T(0)\cdot \overline{T(\infty)} = -1\text{;}\) therefore, we have

\begin{equation*} \frac{pp_a - \lambda pp_a}{p_a - \lambda p} \cdot \frac{\overline{p}-\overline{\lambda}\overline{p_a}}{1-\overline{\lambda}} = -1. \end{equation*}

If we expand this expression and solve it for \(\lambda\overline{\lambda}\) (using the fact that \(p\cdot \overline{p_a} = -1\text{,}\) and \(p \neq p_a\)), we obtain \(\lambda \overline{\lambda} = 1\text{,}\) from which it follows that \(|\lambda| = 1\text{.}\) Thus \(T\) is an elliptic Möbius transformation.
