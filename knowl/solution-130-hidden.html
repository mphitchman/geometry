1. Since segment \(z_0q\) is a radius of the circle, this segment meets the line tangent to the circle at \(q\) at an angle of \(\pi/2\text{.}\) It follows that \(\angle z_0qm_q + \pi/2 + \alpha = \pi\text{,}\) so that \(\angle z_0qm_q = \pi/2 - \alpha\text{.}\) Then, by the right triangle \(\Delta qz_0m_q\) it follows that \(\angle m_qz_0q = \alpha\text{.}\) A similar argument applies for \(\angle pz_0m_p\text{.}\)
  2. If \(q = ke^{i\gamma}\) then \(q^* = \frac{1}{k}e^{i\gamma}\) so \(|q^*-q| = \frac{1}{k}-k = \frac{1-k^2}{k}\text{.}\) Then \(|m_q - q| = \frac{1}{2}|q^*-q|\text{.}\) That is, \(|m_q - q| = \frac{1-k^2}{2k}\text{.}\) If \(p = r\) then \(p^* = \frac{1}{r}\) and the argument above can be used to show that \(|m_p - p| = \frac{1-r^2}{2r}\text{.}\)
  3. In the previous exercise we computed \(\sinh(a)\) and \(\sinh(b)\text{,}\) so
    \begin{align*} \frac{\sinh(a)}{\sin(\alpha)} \amp= \frac{\frac{2r}{1-r^2}}{\frac{1-k^2}{2kR}}\\ \amp= \frac{2r}{1-r^2}\cdot\frac{2kR}{1-k^2}\\ \amp= \frac{4krR}{(1-r^2)(1-k^2)}\text{.} \end{align*}
    And
    \begin{align*} \frac{\sinh(b)}{\sin(\beta)} \amp= \frac{\frac{2k}{1-k^2}}{\frac{1-r^2}{2rR}}\\ \amp= \frac{2k}{1-k^2}\cdot\frac{2rR}{1-r^2}\\ \amp= \frac{4krR}{(1-r^2)(1-k^2)}\text{.} \end{align*}
    Therefore,
    \begin{equation*} \frac{\sinh(a)}{\sin(\alpha)} = \frac{\sinh(b)}{\sin(\beta)}\text{.} \end{equation*}
  4. To see that the ratio \(\sinh(c)/\sin(\gamma)\) must match the common ratio above, note that transformations in hyperbolic geometry preserve distances and angles, so we may transform the triangle above to one in which the length \(c\) is now on the real axis with one end at the origin. The argument above ensures that the ratio \(\sinh(c)/\sin(\gamma)\) then matches one of the other ratios, and so all three agree.
