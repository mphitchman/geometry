Solution 3.5.5.1.

If \(T(z) = (az+b)/(cz+d)\) and \(T(\infty) = \infty\) then \(c = 0\text{,}\) since otherwise \(T(\infty) = a/c \neq \infty\text{.}\) So \(T(z) = (az+b)/d = Az + B\text{.}\) Now \(z = Az + B\) if and only if \(z = B/(1-A)\) so we get a second fixed point if \(A \neq 1\text{.}\) since \(T\) fixes just \(\infty\text{,}\) \(A = 1\text{.}\) This means \(T(z) = 1z + B\) is a translation.
