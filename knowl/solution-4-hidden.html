We build saddleland by cutting a radial slit in a circle (through \(C\) in the following diagram), and adding to it a wedge that matches wedge \(C^\prime OC\) and then taping the two free radial edges.

Consulting the diagram below, notice that \(\theta = 2\pi + \theta^\prime\text{,}\) where \(\theta^\prime = \angle C^\prime O C\) as in the following figure. The angle at \(A\) is \(\alpha\text{,}\) the angle at \(B\) is \(\beta\) and the angle at \(C\) is \(\angle OC^\prime B + \angle OCA = \gamma_1 + \gamma_2 = \gamma\) as in the following figure. Quadrilateral \(ABCC^\prime\) has angle sum \(2\pi\text{,}\) and labeling angles as in the figure, this is written

\begin{equation*} \alpha+\beta+\alpha^\prime+\phi_1+\beta^\prime+\phi_2+\eta_1+\eta_2 = 2\pi \tag{*}\text{.} \end{equation*}

From triangle \(AC^\prime C\text{:}\)

\begin{equation*} \alpha^\prime + \phi_1 + \eta_1 + \eta_2 - \gamma_2 = \pi\text{,} \end{equation*}

or

\begin{equation*} \alpha^\prime+\phi_1 = \pi+\gamma_2-(\eta_1+\eta_2)\text{.} \end{equation*}

From triangle \(BCC^\prime\text{:}\)

\begin{equation*} \beta^\prime + \phi_2 + \eta_1 + \eta_2 - \gamma_1 = \pi\text{,} \end{equation*}

or

\begin{equation*} \beta^\prime + \phi_2 + \eta_1+\eta_2 = \pi+\gamma_1\text{.} \end{equation*}

So (*) becomes

\begin{equation*} \alpha + \beta + (\pi +\gamma_2-\eta_1-\eta_2) + (\pi + \gamma_1) = 2\pi \end{equation*}

which reduces to \(\alpha + \beta + \gamma = \eta_1+\eta_2\text{.}\) Finally, \(\eta_1+\eta_2 = \pi - \theta^\prime\text{,}\) so that \(\eta_1+\eta_2 = \pi - (\theta - 2\pi)\text{.}\) Therefore,

\begin{equation*} \alpha + \beta + \gamma = 3\pi - \theta\text{.} \end{equation*}
