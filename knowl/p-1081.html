Let \(\alpha = \angle T z 0\text{,}\) and \(\beta = \angle z T 0\text{,}\) so that \(\beta = \frac{\pi}{2} - \alpha\text{.}\) Then \(\angle z_a T 0 = \alpha\) and \(\angle T z_a 0 = \beta\text{.}\) Since their three angles are equal, triangles \(\Delta z 0 T\) and \(\Delta T 0 z_a\) are similar. It follows that

\begin{equation*} \frac{|0 - z|}{|0 - T|} = \frac{|0-T|}{|0 - z_a|}\text{,} \end{equation*}

and since \(|0 - T| = 1\)

\begin{align*} |z|\cdot |z_a| \amp= 1\tag{1}\text{.} \end{align*}

But \(z_a = kz\) for some \(k \lt 0\text{,}\) so \(|z_a| = |k||z| = -k|z|\) which, when plugged into equation (1) yields \(-k|z|^2 = 1\text{.}\) Therefore,

\begin{equation*} z_a = \frac{-1}{|z|^2} z\text{.} \end{equation*}
