Item 8.1.1.d.
We offer a brute force approach. Given \(u = a + b\textbf{i} + c\textbf{j} + d\textbf{k}\) and \(v = x + y\textbf{i} + z\textbf{j} + w\textbf{k}\text{,}\)
\begin{align*} uv \amp= (a + b\textbf{i} + c\textbf{j} + d\textbf{k})(x + y\textbf{i} + z\textbf{j} + w\textbf{k})\\ \amp= ax + ay \textbf{i} + az \textbf{j} + aw \textbf{k}\\ \amp~~~+ bx \textbf{i} + by\textbf{i}^2 + bz\textbf{ij} + bw\textbf{ik}\\ \amp~~~+ cx\textbf{j} + cy\textbf{j i}+cz\textbf{j}^2 + cw\textbf{j k}\\ \amp~~~+ dx\textbf{k}+dy{\bf k i} + dz\textbf{k j}+dw\textbf{k}^2\\ \amp= (ax-by-cz-dw)+(ay + bx + cw - dz)\textbf{i} + (az - bw + cx + dy)\textbf{j} + (aw + bz - cy + dx)\textbf{k}\text{,} \end{align*}
so, hold your breath,
\begin{align*} |uv|^2 \amp= (ax-by-cz-dw)^2+(ay + bx + cw - dz)^2 + (az - bw + cx + dy)^2 + (aw + bz - cy + dx)^2\\ \amp= a^2x^2 + b^2y^2 + c^2z^2 + d^2w^2 - 2abxy - 2acxz - 2adxw + 2bcyz+ 2bdyw+2cdzw\\ \amp~~~+ a^2y^2 + b^2x^2 + c^2w^2 + d^2z^2 + 2abxy + 2acyw-2adyz + 2bcxw - 2bdxz-2cdzw \\ \amp~~~+ a^2z^2 + b^2w^2 + c^2x^2+d^2y^2 - 2abzw + 2acxz + 2adyz - 2bcxw- 2bdyw+2cdxy\\ \amp~~~+ a^2w^2+b^2z^2+c^2y^2+d^2x^2+2abzw- 2acyw+2adxw-2bcyz+2bdxz-2cdxy\text{.} \end{align*}
All the cross terms cancel, so that
\begin{align*} |uv|^2 \amp= a^2x^2 + b^2y^2 + c^2z^2 + d^2w^2 + a^2y^2 + b^2x^2 + c^2w^2 + d^2z^2 \\ \amp~~~+ a^2z^2 + b^2w^2 + c^2x^2+d^2y^2+ a^2w^2+b^2z^2+c^2y^2+d^2x^2\\ \amp= a^2(x^2+y^2+z^2+w^2)+b^2(x^2+y^2+z^2+w^2)+c^2(x^2+y^2+z^2+w^2)+d^2(x^2+y^2+z^2+w^2)\\ \amp= (a^2+b^2+c^2+d^2)(x^2+y^2+z^2+w^2)\\ \amp= |u|^2|v|^2\text{,} \end{align*}
and it follows that \(|uv| = |u||v|\text{.}\)
