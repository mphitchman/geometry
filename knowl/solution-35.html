Solution 3.2.1.1.

Translate back to the origin, dilate so the circle has radius 1, invert about the unit circle, dilate back to radius \(r\text{,}\) and translate the origin back to \(z_0\text{.}\) So any point makes this journey:

\begin{align*} z \amp \mapsto z - z_0 \amp \text{(translate)}\\ \amp\mapsto \frac{1}{r}(z - z_0) \amp\text{(stretch)}\\ \amp\mapsto \frac{r}{\overline{z-z_0}} \amp\text{(invert)}\\ \amp\mapsto r\frac{r}{\overline{z-z_0}} \amp\text{(stretch back)}\\ \amp\mapsto \frac{r^2}{\overline{z-z_0}} + z_0. \amp\text{(translate back)}\text{.} \end{align*}
