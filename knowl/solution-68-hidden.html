1. Set \(T(z)=z\) to find the fixed points are \(z = 0,1\text{,}\) so \(T\) has normal form
    \begin{equation*} \frac{T(z)-1}{T(z)-0} = \lambda \frac{z-1}{z-0}\text{.} \end{equation*}
    Noting that \(T(2) = 2/3\) we can solve for \(\lambda = -1 = e^{i\pi}\text{,}\) so \(T\) is a elliptic and rotates points about type II clines of 0 and 1. In the diagram below, we have plotted some type II clines of the fied points, marked \(p\) and \(q\text{.}\) Two other points \(z\) and \(w\) have been plotted along with their image points (on the same type II cline of \(p\) and \(q\) as the input). We remark that this map is also an involution (\(T \circ T\) is the identity).
  2. Fixed points are \(z = 0,i\text{,}\) and \(T\) has normal form
    \begin{equation*} \frac{T(z)-i}{T(z)-0} = \lambda \frac{z-i}{z -0}\text{.} \end{equation*}
    Noting that \(T(1)=-1\text{,}\) \(\lambda = i = e^{i\pi/2}\) and \(T\) is again elliptic, rotating points about type II clines of 0 and \(i\text{.}\)
  3. Setting \(T(z)=z\) yields a quadratic equation:
    \begin{align*} 3iz-5 \amp = z(z-i) \\ 0 \amp = z^2 - 4iz + 5\\ 0 \amp = (z-5i)(z+i) \text{.} \end{align*}
    So the fixed points are \(z = 5i, -i\) producing normal form
    \begin{equation*} \frac{T(z)-5i}{T(z)+i} = \lambda \frac{z-5i}{z+i}\text{.} \end{equation*}
    From the original description of the map, \(T(\infty)=3i\text{,}\) and plugging \(z=\infty\) into the normal form yeilds
    \begin{equation*} \frac{-2i}{4i} = \lambda \cdot 1\text{.} \end{equation*}
    Thus, \(\lambda = -\frac{1}{2}\text{,}\) a real number so the map is hyperbolic and preserves type I clines of the two fixed points.
