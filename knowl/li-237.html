Item 5.5.5.c.
To compute the side lengths, let
\begin{equation*} \boldsymbol{r}_1(t) = t + mi ~~\text{ for }~ a \leq t \leq b \end{equation*}
and
\begin{equation*} \boldsymbol{r}_2(t) = t + ni ~~\text{ for }~~ a \leq t \leq b \end{equation*}
describe the two horizontal segments in the block. Following the statement of the problem, set \(c = {\cal L}(\boldsymbol{r}_1)\) and \(d = {\cal L}(\boldsymbol{r}_2)\text{.}\) Note that \(\boldsymbol{r}_1^\prime(t) = \boldsymbol{r}_2^\prime(t) = 1\text{,}\) so
\begin{equation*} c = \int_a^b \frac{1}{m}~dt = \frac{b-a}{m} \end{equation*}
and
\begin{equation*} d = \int_a^b \frac{1}{n}~dt = \frac{b-a}{n}\text{.} \end{equation*}
Let \(x = \ln(n/m)\) denote the common length from part (b) so that \(e^x = \frac{n}{m}\text{.}\) Then
\begin{align*} d e^x \amp= \frac{b-a}{n}\cdot \frac{n}{m}\\ \amp= \frac{b-a}{m}\\ \amp= c \end{align*}
as desired.
