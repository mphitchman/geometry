Solution 4.1.4.1.

This is a group. The identity is in there: let \(k = 1\text{.}\) Closure: if \(T_1(z) = k_1z\) and \(T_2(z)=k_2z\) then \(T_2\circ T_1(z) = (k_2k_1)z\text{.}\) Inverses: Given \(T(z) = kz\text{,}\) \(k > 0\) so\(1/k > 0\) as well, and \(T^{-1}(z) = \frac{1}{k}z\text{.}\)
