Convince yourself that no “closed sets” of corners, such as we saw in the Poincaré dodecahedral space, occur in the Seifert-Weber space. That is, if we start at a corner, find the 3 corners identified with the initial one via the three face identifications, and proceed to find each of these corners' three neighbors, and so on, we will eventually gather all 20 corners into a single point.
