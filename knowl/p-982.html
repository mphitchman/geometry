Each \(z\) in \(\mathbb{C}^+\) has a unique antipodal point, \(z_a\text{,}\) given as follows:

\begin{equation*} z_a =\begin{cases}-1/\overline{z} \amp \text{ if \(z \neq 0, \infty\); } \\ \infty \amp \text{ if \(z = 0\); } \\ 0 \amp \text{ if \(z=\infty\). } \end{cases} \end{equation*}
