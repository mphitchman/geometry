Item 5.4.12.b.
If \(q = ke^{i\gamma}\) then \(q^* = \frac{1}{k}e^{i\gamma}\) so \(|q^*-q| = \frac{1}{k}-k = \frac{1-k^2}{k}\text{.}\) Then \(|m_q - q| = \frac{1}{2}|q^*-q|\text{.}\) That is, \(|m_q - q| = \frac{1-k^2}{2k}\text{.}\) If \(p = r\) then \(p^* = \frac{1}{r}\) and the argument above can be used to show that \(|m_p - p| = \frac{1-r^2}{2r}\text{.}\)
