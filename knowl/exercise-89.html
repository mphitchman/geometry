Exercise 8.

Rotational geometry is the geometry \((\mathbb{C}, {\cal R})\) where \({\cal R}\) is the group of rotations about the origin. That is

\begin{equation*} {\cal R} = \{ R_\theta(z) = e^{i\theta}z ~|~ \theta \in \mathbb{R}\}\text{.} \end{equation*}
  1. Prove that \(\cal R\) is a group of transformations.
  2. Is \({\cal D} = \{\)all lines in \(\mathbb{C}\}\) an invariant set in rotational geometry? Is it a minimally invariant set?
  3. Find a minimally invariant set of rotational geometry that contains the circle \(|z-(2+i)| = 4\text{.}\)
  4. Is \((\mathbb{C}, {\cal R})\) homogeneous? Isotropic?
