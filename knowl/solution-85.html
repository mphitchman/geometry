Solution 4.1.3.1.

Recall, a Euclidean transformation has the form \(T(z) = e^{i\theta}z + b\text{.}\)

Identity: \(T(z) = z\) is the Euclidean transformation with \(\theta = 0, b = 0\text{.}\)

Closure: If \(T(z) = e^{i\theta}z + b\) and \(S(z) = e^{i\beta}z + c\) then

\begin{align*} T\circ S(z) \amp = e^{i\theta}[e^{i\beta}z+c]+b \\ \amp = e^{i(\theta+\beta)}z+[e^{i\theta}c + b] \end{align*}

is a Euclidean transformation.

Inverses: Given \(T(z) = e^{i\theta}z + b\text{,}\) \(T^{-1}(z)=e^{-i\theta}z - e^{-i\theta}b\text{.}\)
