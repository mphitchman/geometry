Item 8.1.1.b.

From \(\textbf{i j} = \textbf{k}\text{,}\) \(\textbf{i}^2\textbf{j} = \textbf{ik}\text{,}\) which implies \(\textbf{i k} = -\textbf{j}\text{.}\)

Similarly, \(\textbf{k i } = \textbf{j}\) implies \(\textbf{k i}^2 = \textbf{ji}\text{,}\) which implies \(\textbf{j i} = -\textbf{k}\text{.}\)

Finally, \(\textbf{i j} = \textbf{k}\) implies that \(\textbf{i j}^2 = {\bf kj}\text{,}\) which implies \(\textbf{k j} = -\textbf{i}\text{.}\)
