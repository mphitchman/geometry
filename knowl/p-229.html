We begin with a definition.

Definition 3.1.1.

Given two sets \(A\) and \(B\text{,}\) a function \(f:A \to B\) is called one-to-one (or 1-1) if whenever \(a_1 \neq a_2\) in \(A\text{,}\) then \(f(a_1) \neq f(a_2)\) in \(B\text{.}\) The function \(f\) is called onto if for any \(b\) in \(B\) there exists an element \(a\) in \(A\) such that \(f(a) = b\text{.}\) A transformation on a set \(A\) is a function \(T: A \to A\) that is one-to-one and onto.
