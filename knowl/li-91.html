Item 3.5.2.c.
Setting \(T(z)=z\) yields a quadratic equation:
\begin{align*} 3iz-5 \amp = z(z-i) \\ 0 \amp = z^2 - 4iz + 5\\ 0 \amp = (z-5i)(z+i) \text{.} \end{align*}
So the fixed points are \(z = 5i, -i\) producing normal form
\begin{equation*} \frac{T(z)-5i}{T(z)+i} = \lambda \frac{z-5i}{z+i}\text{.} \end{equation*}
From the original description of the map, \(T(\infty)=3i\text{,}\) and plugging \(z=\infty\) into the normal form yeilds
\begin{equation*} \frac{-2i}{4i} = \lambda \cdot 1\text{.} \end{equation*}
Thus, \(\lambda = -\frac{1}{2}\text{,}\) a real number so the map is hyperbolic and preserves type I clines of the two fixed points.
