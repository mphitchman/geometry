If \(b \neq 0\text{,}\) the translation \(T_b\) of \(\mathbb{C}\) has no fixed points. Rotations of \(\mathbb{C}\) and dilations of \(\mathbb{C}\) have a single fixed point, and the general linear transformation \(T(z) = az + b\) has one fixed point as long as \(a \neq 1\text{.}\) To find this fixed point, solve

\begin{equation*} z = az + b \end{equation*}

for \(z\text{.}\) For instance, the fixed point of the transformation \(T(z) = 2z + (4-i)\) of ExampleÂ 3.1.10 is found by solving \(z = 2z + 4 - i\text{,}\) for \(z\text{,}\) which yields \(z =-4 + i\text{.}\) So, while the map \(T(z) = 2z + (4-i)\) moves the disk \(D\) in the example to the disk \(T(D)\text{,}\) the point \(-4+i\) happily stays where it is.
