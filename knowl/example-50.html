Example 5.5.7 The length of a curve

To find the length of the horizontal curve \(\boldsymbol{r}(t) = t + ki\) for \(a \leq t \leq b\text{,}\) note that \(\boldsymbol{r}^\prime(t) = 1\) and \(\text{ Im}(\boldsymbol{r}(t)) = k\text{.}\) Thus,

\begin{equation*} {\cal L}(\boldsymbol{r}) = \int_a^b \frac{1}{k}~dt = \frac{b-a}{k}. \end{equation*}
