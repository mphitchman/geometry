Definition4.1.12

A metric for a geometry \((S,G)\) is an invariant function \(d:S \times S \to \mathbb{R}\) mapping each ordered pair \((x,y)\) of elements from \(S\) to a real number such that

  1. \(d(x,y) \geq 0\) for all \(x,y \in S\) and \(d(x,y) = 0\) if and only if \(x = y\text{.}\)

  2. \(d(x,y) = d(y,x)\) for all \(x,y \in S\text{.}\)

  3. (Triangle inequality) \(d(x,z) \leq d(x,y) + d(y,z)\) for all \(x,y,z \in S\text{.}\)
