Solution 8.2.3.1.

The transformation \(T_a\) in Example 7.7.11 consists of reflection about the first \(a\) edge followed by reflection about the line \(m\text{,}\) as indicated in the accompanying diagram.

The map moves the point \(q = re^{7\pi/4 i}\) to \(T_a(q)= r\) and \(p = re^{6\pi/4 i}\) to \(T_a(p)=re^{\pi/4 i}\text{.}\) Using the working distance formula, one checks that

\begin{equation*} d_H(q,T(q)) \neq d_H(p,T(p))\text{.} \end{equation*}

In fact, \(r = (1/2)^{(1/4)}\) by Exercise 5.4.9, so \(d_H(q,T(q))\approx 3.057\) and \(d_H(p,T(p))\approx 4.741\text{.}\)
