Solution 5.4.11.1.
  1. Suppose \(x,y\) are positive real numbers.
    \begin{align*} \cosh(\ln(x/y)) \amp= \frac{e^{\ln(x/y)} + e^{-\ln(x/y)}}{2}\\ \amp= \frac{\frac{x}{y}+\frac{y}{x}}{2}\\ \amp= \frac{x^2 + y^2}{2xy}\text{.} \end{align*}
    Similarly,
    \begin{align*} \sinh(\ln(x/y)) \amp= \frac{e^{\ln(x/y)} - e^{-\ln(x/y)}}{2}\\ \amp= \frac{\frac{x}{y}-\frac{y}{x}}{2}\\ \amp= \frac{x^2 - y^2}{2xy}\text{.} \end{align*}
  2. Let \(s = |1 - \overline{p}q|\) and \(t = |q - p|\text{.}\) Then, letting \(c = d_H(p,q)\text{,}\)
    \begin{align*} \cosh(c) \amp= \cosh\bigg(\ln\bigg(\frac{s + t}{s - t}\bigg)\bigg)\\ \amp= \frac{(s+t)^2 + (s-t)^2}{2(s+t)(s-t)}\tag{by part (a)}\\ \amp= \frac{2s^2 + 2t^2}{2(s^t - t^2)}\\ \amp= \frac{s^2 + t^2}{s^2 - t^2}\text{.} \end{align*}
    Now,
    \begin{equation*} s^2 = (1-\overline{p}q)(1-p\overline{q}) = 1 - \overline{p}q-p\overline{q}+|pq|^2 \end{equation*}
    and
    \begin{equation*} t^2 = (q-p)(\overline{q}-\overline{p}) = |q|^2 - p\overline{q}-\overline{p}q+|p|^2 \end{equation*}
    so that
    \begin{align*} s^2 + t^2 \amp= 1 + |p|^2 + |q|^2 + |pq|^2 - 2(\overline{p}q+p\overline{q})\\ \amp= (1 + |p|^2)(1 + |q|^2) - 4\text{Re}(p\overline{q})\text{.} \end{align*}
    One also checks that
    \begin{equation*} s^2 - t^2 = (1 - |p|^2)(1-|q|^2)\text{,} \end{equation*}
    so that
    \begin{equation*} \cosh(c) = \frac{(1 + |p|^2)(1 + |q|^2) - 4\text{Re}(p\overline{q})}{(1 - |p|^2)(1-|q|^2)}\text{.} \end{equation*}
  3. Since
    \begin{equation*} a = d_H(0,r) = \ln\bigg(\frac{1+r}{1-r}\bigg)\text{,} \end{equation*}
    it follows from part (a) that
    \begin{align*} \sinh(a) \amp= \frac{(1+r)^2-(1-r)^2}{2(1+r)(1-r)}\\ \amp= \frac{2r}{1-r^2}\text{.} \end{align*}
    Using part (a) again, we see
    \begin{align*} \cosh(a) \amp= \frac{(1+r)^2+(1-r)^2}{2(1+r)(1-r)}\\ \amp= \frac{1+r^2}{1-r^2}\text{.} \end{align*}
    Turning to the other side,
    \begin{equation*} b = d_H(0,ke^{i\gamma}) = \ln\bigg(\frac{1+k}{1-k}\bigg)\text{,} \end{equation*}
    so
    \begin{equation*} \sinh(b) = \frac{2k}{1-k^2}~\text{ and }~\cosh(b) = \frac{1+k^2}{1-k^2} \end{equation*}
    by applying the same argument.
  4. Apply the result in part (b), where \(p = r\) and \(q = ke^{i\gamma}\) so that \(|p| = r\) and \(|q| = k\text{.}\)
    \begin{align*} \cosh(c) \amp= \frac{(1+r^2)(1+k^2)- 4\text{Re}(rke^{-i\gamma})}{(1-r^2)(1-k^2)}\\ \amp= \frac{(1+r^2)(1+k^2)- 4rk\cos(-\gamma)}{(1-r^2)(1-k^2)}\\ \amp= \frac{1+r^2}{1-r^2}\cdot \frac{1+k^2}{1-k^2}- \frac{2r}{1-r^2}\cdot \frac{2k}{1-k^2}\cos(\gamma)\tag{}\\ \amp= \cosh(a)\cosh(b)-\sinh(a)\sinh(b)\cos(\gamma)\text{.} \end{align*}
  5. The result holds for general hyperbolic triangles because any triangle is congruent to one with a vertex at the origin and one leg on the positive real axis, and angles and distances are preserved under transformations in \(\cal H\text{.}\)
