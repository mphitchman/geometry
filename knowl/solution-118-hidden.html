The set of all points in \(\mathbb{D}\) a hyperbolic distance \(r\) from \(z_0\) can be found as follows. Suppose first \(z_0\) is at the origin in \(\mathbb{D}\text{.}\) Any point \(z\) in \(\mathbb{D}\) has distance to the origin equal to

\begin{equation*} d_H(0,z)=\ln\left(\frac{1+|z|}{1-|z|}\right)\text{.} \end{equation*}

Thus, \(z\) is on \(C\) if and only if

\begin{equation*} \ln\bigg(\frac{1+|z|}{1-|z|}\bigg) = r \end{equation*}

which is true if and only if

\begin{equation*} |z| = \frac{e^r - 1}{e^r + 1}\text{.} \end{equation*}

This collection of points is precisely a Euclidean circle centered at the origin.

For an arbitrary point \(z_0\) in \(\mathbb{D}\text{,}\) first transform it to the origin to see that the image of \(C\) under this transformation is a Euclidean circle centered at 0. The inverse transformation then takes this circle back to \(C\text{.}\) This inverse transformation must preserve clines since it is a MÃ¶bius transformation. This map also sends \(\mathbb{D}\) to itself, since it is an element of \(\cal H\text{,}\) which means that no point on the circle centered at the origin can be mapped to \(\infty\text{.}\) In particular, \(C\) is a circle, not a line.
