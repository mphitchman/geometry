Solution 3.4.16.1.

First, translate by \(-z_2\text{.}\) This map sends \(z_2\) to 0, keeps \(z_3=\infty\text{,}\) and sends \(z_1\) to \(z_1-z_2\text{.}\)

Second, rotate about the origin so that \(z_1-z_2\) moves to the positive real axis (its new location will be \(|z_1-z_2|\text{.}\) This map fixes 0 and \(\infty\text{.}\)

Third, dilate by \(\frac{1}{|z_1-z_2|}\text{.}\) This map fixes 0 and \(\infty\) and moves \(|z_1-z_2|\) to 1.

Composing these three maps has the desired effect: \(z_1 \mapsto 1\text{,}\) \(z_2 \mapsto 0\text{,}\) and \(z_3 \mapsto \infty\text{.}\) Furthermore, translations, rotations, and dilations each can all be achieved by the composition of two inversions, so the map we have built is indeed the composition of an even number of inversions.
