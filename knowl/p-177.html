Since \(z\) is on the unit circle, \(|z| = 1\) (and \(|z|^2 = 1\)). Then,

\begin{align*} \angle b z a \amp = \text{arg}\bigg(\frac{1-z}{-1-z}\bigg)\\ \amp =\text{arg}\bigg(\frac{(1-z)\overline{(-1-z)}}{|-1-z|^2}\bigg)\\ \amp = \text{arg}(-1+z-\overline{z}+|z|^2)-\text{arg}(|-1-z|^2)\\ \amp = \text{arg}(z-\overline{z})-\text{arg}(|-1-z|^2)\text{.} \end{align*}

Since \(|-1-z|^2\) is a real number, \(\text{arg}(|-1-z|^2) = 0\text{,}\) and so \(\angle -1 z 1\) equals \(\text{arg}(z-\overline{z})\text{.}\) But \(z - \overline{z} = 2\text{Im}(z)i\) is on the positive imaginary axis, so this argument equals \(\pi/2\text{.}\) That is, \(\angle bza = \pi/2\text{.}\)
